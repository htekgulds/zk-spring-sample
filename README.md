7.0.3

1. Spring Web - 4.1.7
2. Spring Data JPA - 1.8.0
3. Spring Security - 4.0.1
4. Hibernate Core - 4.3.10
5. Hibernate Envers for Auditing - 4.3.10
6. Spring Auditing - 4.1.7
8. Postgresql - 9.4
